请将 Google Service Account 的 JSON 凭证文件放在此目录下，命名为 google_service_account.json。

获取步骤：
1. 访问 https://console.cloud.google.com/
2. 创建/选择项目
3. 启用 Google Sheets API 和 Google Drive API
4. APIs & Services → Credentials → Create Credentials → Service Account
5. 下载 JSON 密钥文件
6. 将文件重命名为 google_service_account.json 并放入此目录

注意：此目录和凭证文件已被 .gitignore 排除，不会提交到版本控制。
